lcd_bias
========

LCD bias voltage generator circuit for using 5V character LCDs on 3 volts

Please visit the project page at http://jap.hu/electronic/lcd_at_3_volts.html

This is the eagle v6 schematic and board layout for the charge pump circuit.
You can buy the assembled and tested board in my Tindie store
at https://www.tindie.com/stores/jap/
